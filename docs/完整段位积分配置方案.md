# 完整段位积分配置方案

## 积分获取规则回顾

### 订单完成获得积分
- **基础比例**：1元 = 1积分
- **段位倍数**：
  - 黑铁/青铜：1.0x（1元=1积分）
  - 白银：1.2x（1元=1.2积分）
  - 黄金：1.5x（1元=1.5积分）
  - 白金及以上：2.0x（1元=2积分）

### 评价获得积分
- **基础评价**：50积分
- **带图评价**：+20积分（共70积分）
- **首次评价**：+30积分（共80积分）
- **首次带图评价**：100积分

### 评价采纳获得积分
- **采纳奖励**：200积分

## 配置说明

### 设计原则
1. **合理梯度**：段位之间积分差距逐步增大
2. **折扣匹配**：高段位提供更高折扣，匹配更高的积分要求
3. **倍数效应**：高段位积分倍数更高，形成正向循环
4. **用户激励**：让用户有持续的动力升级

### 段位分组说明

## 积分获取速度分析

假设平均订单金额为200元：

1. **黑铁/青铜段位（1.0x）**：
   - 每次订单：200积分
   - 达到黑铁二（100积分）：0.5次订单
   - 达到青铜三（1500积分）：7.5次订单

2. **白银段位（1.2x）**：
   - 每次订单：240积分
   - 达到白银一（2200积分）：约9次订单（从0开始）
   - 达到白银三（4000积分）：约17次订单（从0开始）

3. **黄金段位（1.5x）**：
   - 每次订单：300积分
   - 达到黄金一（5500积分）：约23次订单（从0开始）
   - 达到黄金三（10000积分）：约42次订单（从0开始）

4. **白金及以上段位（2.0x）**：
   - 每次订单：400积分
   - 达到白金一（15000积分）：约38次订单（从0开始）
   - 达到赋能（250000积分）：约625次订单（从0开始）
